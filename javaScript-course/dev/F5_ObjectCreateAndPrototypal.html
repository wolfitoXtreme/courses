F5_ObjectCreateAndPrototypal

  • Apart from function constructors, JavaScript has another way to create objects, is the Object.create (Classic Inheritance).
  • In ECMA6/ES6 there is another way to create objects, is classes, they do just the same. They are objects themselves, and serve as a way to create objects via the new keyword. They also have extends, which is used to change the prototype of the class (object).
