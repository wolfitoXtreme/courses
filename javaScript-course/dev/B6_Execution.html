B6_Execution

  • Single threaded execution: one line at the time.
  • Synchronous: one execution line at the time in order.
  • ASynchronous: More that one execution line at the time in order.
  • Invocations: Running a function, in javaScript using parenthesis '()'.
  • Execution Stack: Every function when called, creates a new Execution Context, and goes to the top of the Execution stack.
