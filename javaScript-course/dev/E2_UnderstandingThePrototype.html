E2_UnderstandingThePrototype

  • All objects (including functions) in JavaScript have a prototype property.
  • All properties are references to the proto{} object.
  • All prototype objects can also reference to yet another prototype object.
  • This relation between objects and prototypes is called Prototype Chain.
  • Independent Objects can share prototypes between them.
  • In JavaScript Everything is an Object or a Primitive, and everything has a base prototype object down the prototype chain.
